# Retail SQL Data Analyst Agent

A LangGraph-based AI agent that answers natural-language retail business
questions by generating safe, read-only SQL against a MySQL database,
and replies with concise business summaries.

> 🚧 Work in progress — built step by step as part of the
> Agentic AI Explorer (Level 2) exercise.

## Tech Stack
- Python 3.12, managed with [uv](https://docs.astral.sh/uv/)
- LangGraph (agent workflow), Groq (free LLM)
- MySQL 8 (data source)
